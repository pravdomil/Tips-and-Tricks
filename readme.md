# Ubuntu server
Initial setup
```
	ssh‑copy‑id root@$IP
	ssh root@$IP
	update‑locale LC_ALL=en_US.UTF‑8 LANG=en_US.UTF‑8
	logout
	ssh root@$IP
	apt update
	apt upgrade
	reboot
```

SSH tunel  
`ssh -L LOCAL_PORT:DEST:DEST_PORT TUNEL_USER@TUNEL_SERVER -p PORT`

Create symlink  
`ln -s SOURCE SYMLINK`
